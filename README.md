# Simple C++ Calculator

## Overview
This is a **console-based calculator** written in C++ that performs basic arithmetic operations. It allows the user to calculate **addition, subtraction, multiplication, division (quotient), and modulus (remainder)**. The program runs in a loop until the user chooses to exit.

---

## Features
- Perform **Addition (`+`)**
- Perform **Subtraction (`-`)**
- Perform **Multiplication (`*`)**
- Perform **Division (`/`)** – displays quotient; handles division by zero.
- Perform **Modulus (`%`)** – displays remainder; handles division by zero.
- **Exit option** by entering `0` as the operator.
- Handles invalid operator inputs gracefully.

---

## How to Use
1. Compile the program using a C++ compiler:
   g++ calculator.cpp -o calculator
2.Run the compiled program:
   ./calculator.exe
3.Follow the prompts:
Enter the first number

Enter the second number

Enter the operator (+, -, *, /, %) or 0 to exit.

The program will display the result and ask for new inputs until you choose to exit.
